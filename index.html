Mis Servicios

Machine Learning

Creación de modelos predictivos, gracias a la integración de diferentes disciplinas y estrategias, que van de la mano con algoritmos haciendo del poder de computo con la que contamos hoy en día. Todo esto enfocado en resolver problemas de clasificación, de tal forma que, todo esto se haga de manera automática con el paso del tiempo, y como una consecuencia, realizar predicciones con base a patrones para generar tendencias acordes al comportamiento histórico y posiblemente, predecir ciertos eventos en el futro. Todo esto, basado en la información que se capta de las variables de los distintos sistemas.

-Clasificación ¿Es cáncer o no es cáncer? ¿La imagen pertenece a un ave, perro o gato? ¿A que segmento de clientes pertenece determinado usuario? -Regresión Predecir el precio del dólar para el mes siguiente El total de calorías de una comida dados sus ingredientes La ubicación mas probable de determinado objeto dentro de una imagen. -Clustering Identificar productos similares para un sistema de recomendación Descubrir el sitio ideal para ubicar paradas de autobuses según la densidad de población Segmentar imágenes según patrones de colores y geometrías.
